## 🚀 Key Features

* **Real-Time Data Ingestion**: Secure collection of sensor values and GPS coordinates via a dedicated HTTPS REST API.
* **Dynamic Dashboards**: Interactive data visualization using **Chart.js** for sensor trends and **Leaflet.js** for real-time GPS tracking on maps.
* **Smart Alert System**: Automated anomaly detection based on user-defined thresholds (Min, Max, Delay) with real-time status updates (*Warning*, *Alert*, *In-bound*).
* **Administrative Interface**: Complete management of users (RBAC), IoT devices, and physical sensor associations.
* **Advanced Reporting**: Capability to save custom analysis reports and export data in **CSV** (logs) or **PNG** (graphs) formats.

---

## 🛠 Tech Stack

* **Backend**: Node.js & Express.js.
* **Frontend**: Vanilla JavaScript (Single Page Application architecture).
* **Database**: MariaDB (optimized for time-series sensor data).
* **Hardware / Simulators**: Physical **ESP32** (C code) and custom high-volume C simulators developed for testing.
* **Infrastructure**: Deployed on **AWS** (Amazon Linux) using automated bash scripts.

## 🚀 Key Features

* [cite_start]**Real-Time Data Ingestion**: Secure collection of sensor values and GPS coordinates via a dedicated HTTPS REST API[cite: 120, 199, 210, 317].
* [cite_start]**Dynamic Dashboards**: Interactive data visualization using **Chart.js** for sensor trends and **Leaflet.js** for real-time GPS tracking on maps[cite: 184, 185, 327, 499].
* [cite_start]**Smart Alert System**: Automated anomaly detection based on user-defined thresholds (Min, Max, Delay) with real-time status updates (*Warning*, *Alert*, *In-bound*)[cite: 267, 342, 346, 348].
* [cite_start]**Administrative Interface**: Complete management of users (RBAC), IoT devices, and physical sensor associations[cite: 94, 130, 265, 357, 362].
* [cite_start]**Advanced Reporting**: Capability to save custom analysis reports and export data in **CSV** (logs) or **PNG** (graphs) formats[cite: 110, 124, 337, 339, 372].

## 🛠 Tech Stack

* [cite_start]**Backend**: Node.js & Express.js[cite: 166, 169].
* [cite_start]**Frontend**: Vanilla JavaScript (Single Page Application architecture)[cite: 182, 273].
* [cite_start]**Database**: MariaDB (optimized for time-series sensor data)[cite: 177, 206, 562].
* [cite_start]**Hardware/Simulators**: Physical **ESP32** (C code) and custom high-volume C simulators developed for testing[cite: 190, 196, 318, 610].
* [cite_start]**Infrastructure**: Deployed on **AWS** (Amazon Linux) using automated bash scripts[cite: 392, 395, 552].
